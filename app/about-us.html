We love open source

We love coding. And we also believe in open source. That's why we contribute to many projects with people from all over the world to push things forward. Here's what we've contributed to.

AngularJS is a structural framework for dynamic web apps. It lets you use HTML as your template language and lets you extend HTML's syntax to express your application's components clearly and succinctly.

angular-translate is an Angular module built by thoughtram that extends the frameworks internationalization and localization capabilities with dedicated components like services and directives.

Nickel.rs is a web application framework made by thoughtram for the Rust programming language that comes with express-like APIs and features like application routing, templating and JSON handling.
