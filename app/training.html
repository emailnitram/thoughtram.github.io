Training

We regularly organize workshops, webinars or even free training days near you. Here's our portfolio of workshops we offer. You might be interested in one of these.

  • Git Master Class

    Git is a free and open source distributed version control system. It's invention was a blessing to the whole software world. We use Git on our daily basis to be as productive as possible, especially when working in larger teams.

    Attending our Git Master Class gives you a deep understanding of how Git works and what tools it comes with to make working with code a lot easier.

    Learn more →

  • Angular Master Class

    AngularJS is a JavaScript tool set that changes the way we build web applications. We have over 3 years experience with AngularJS, we've built widely used modules for it and are also active contributors to the project.

    In our Angular Master Class attendees learn everything they need to know to build full fledged single page applications with AngularJS.

    Learn more →
